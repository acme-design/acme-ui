Switch 开关

基础用法

基本用法，状态切换时会触发事件。

复制
  1<Switch />
>

大小

想用一些优雅的小开关？ 我们提供了 size 这个属性供您调整。

复制
  1<Switch defaultChecked={true} />
  2<Switch size="small" />
>

禁用状态

禁用开关

复制
  1<Switch disabled />
  2<Switch defaultChecked={true} disabled />
  3<Switch size="small" disabled />
  4<Switch size="small" disabled defaultChecked={true} />
>

加载中

标识开关操作仍在执行中

复制
  1<Switch loading />
  2<Switch defaultChecked loading />
  3<Switch size="small" loading />
  4<Switch size="small" defaultChecked loading />
>

受控

复制
  1() => {
  2  const [checked, setChecked] = React.useState(false)
  3  const [loading, setLoading] = React.useState(false)
  4  const onChange = (check, e) => {
  5    setLoading(true)
  6    setTimeout(() => {
  7      setChecked(check)
  8      setLoading(false)
  9    }, 1000)
 10  }
 11  return <Switch checked={checked} onChange={onChange} loading={loading} />
 12}
>

API

通过设置不同的属性展示不同的开关状态

参数(*为必填)说明类型默认值
classNameSwitch 样式string
sizeSwitch 大小 "default" | "small"default
checked当前是否选中boolean
defaultChecked默认选中状态boolean false
disabled是否禁用boolean false
inputRef真实的input元素((instance: HTMLInputElement | null) => void) | RefObject<HTMLInputElement> | null null
loading是否处于加载状态boolean false
onChange状态切换事件 @param checked @param event((event: ChangeEvent<Element>) => void)
