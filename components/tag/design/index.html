API

参数(*为必填)说明类型默认值
colortag 文字颜色string #0043CE
backgroundColortag 背景颜色string #D0E2FF
modetag 的模式 圆角或方形 "default" | "rect"default
closable标签是否可以关闭boolean false
onClose关闭事件((e: MouseEvent<Element, MouseEvent>) => void)
className组件样式string
