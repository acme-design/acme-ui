全选 （包含子集）

复选框可以与其他复选框具有父子关系。

复制
  1() => {
  2  const options = ["A", "B", "C"];
  3  const [horizontalValues, setHorizontalValues] = React.useState([]);
  4  const [horizontalIndeterminate, setHorizontalIndeterminate] =
  5    React.useState(false);
  6  const [horizontalCheckedAll, setHorizontalCheckedAll] =
  7    React.useState(false);
  8  const horizontalGroupChange = (values) => {
  9    setHorizontalValues(values);
 10    setHorizontalIndeterminate(values.length > 0 && values.length < 3);
 11    setHorizontalCheckedAll(values.length === 3);
 12  };
 13  const horizontalAllChange = (e) => {
 14    const checked = e.target.checked;
 15    setHorizontalCheckedAll(checked);
 16    setHorizontalIndeterminate(false);
 17    setHorizontalValues(checked ? options : []);
 18  };
 19  return (
 20    <div>
 21      <Checkbox
 22        style={{ margin: "10px 0" }}
 23        indeterminate={horizontalIndeterminate}
 24        checked={horizontalCheckedAll}
 25        onChange={horizontalAllChange}
 26      >
 27        全选
 28      </Checkbox>
 29      <Checkbox.Group
 30        values={horizontalValues}
 31        onChange={horizontalGroupChange}
 32        layout="horizontal"
 33        style={{ marginLeft: "20px" }}
 34      >
 35        {options.map((item) => (
 36          <Checkbox
 37            key={item}
 38            value={item}
 39            style={{ marginRight: "30px" }}
 40          >
 41            选项{item}
 42          </Checkbox>
 43        ))}
 44      </Checkbox.Group>
 45    </div>
 46  );
 47};
 48
>

API

通过设置不同的属性展示不同的开关状态

参数(*为必填)说明类型默认值
className最外层元素样式string
childrenlabel内容ReactNode
checked是否选中boolean
defaultChecked初始是否选中boolean
indeterminate半选状态，只控制样式boolean
valuecheckbox 值string | number
onChangecheckbox 状态改变((e: ChangeEvent<Element>) => void)
required是否必填boolean
disabled禁用状态boolean
labelPlacement标签所在位置 "top" | "left" | "right" | "bottom"
Input输入框Form Description表单描述
