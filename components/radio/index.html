Radio单选框

基础用法

由于选项默认可见，不宜过多，若选项过多，建议使用 Select 选择器

复制
  1() => {
  2  const [checkedValue, setCheckedValue] = React.useState(false);
  3  const onChange = (e) => {
  4    const checked = e.target.checked;
  5    setCheckedValue(checked);
  6  };
  7  return (
  8    <div>
  9      <Radio onChange={onChange} checked={checkedValue} label="单选" />
 10    </div>
 11  );
 12};
 13
>

API

参数(*为必填)说明类型默认值
className自定义类名string
stylestyle样式CSSProperties
value当前值string | number
checked是否选中boolean
label单选标签ReactNode
labelPlacement标签位置 "top" | "left" | "right" | "bottom"right
defaultChecked是否选中，非受控属性boolean
disabled是否禁用boolean
onChange事件变化方法((e: ChangeEvent<HTMLInputElement>) => void)
inline当前Radio是否为行内元素boolean
