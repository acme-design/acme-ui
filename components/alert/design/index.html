API

通过设置不同的属性展示不同的开关状态

参数(*为必填)说明类型默认值
className外层class名string
typealert 类型 "error" | "success" | "warning" | "info"info
align文案位置 "left" | "right" | "center"left
closable是否显示关闭按钮boolean
onClosealert 关闭事件((e: MouseEvent<Element, MouseEvent>) => void)
titlealert 头部内容ReactNode
icon自定义iconReactNode
showIcon是否展示默认iconboolean
action自定义操作ReactNode
Select选择器Modal模态框
