API

通过设置不同的属性展示不同状态

参数(*为必填)说明类型默认值
className样式string
children*表单元素ReactNode
id表单元素的idstring
required是否必填boolean
label标题ReactNode
labelFor提供给label的htmlFor，若传了id属性则不需要传该参数string
description描述ReactNode
hint提示文案ReactNode
hintId提供给hint的idstring
status状态 "error" | "success" | "warning"
disabled禁用状态boolean
fullWidth是否撑满父元素boolean
Form Hint表单提示Alert警告提示
