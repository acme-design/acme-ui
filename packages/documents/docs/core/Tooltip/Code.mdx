## 触发方式
3种触发方式：鼠标移入，点击，获得输入焦点

<Playground>
    <div style={{display: 'flex', justifyContent: 'space-between', marginBottom: '100px'}}>
        <Tooltip content="提示文字" trigger="hover">
            <Button>鼠标移入</Button>
        </Tooltip>
        <Tooltip content="提示文字" trigger="click">
            <Button>点击</Button>
        </Tooltip>
        <Tooltip content="这里是提示文字" trigger="focus">
            <Input placeholder="请输入内容" />
        </Tooltip>
    </div>
</Playground>

## 可变的宽度
为了保证可阅读星，默认将较长的文字折行

            默认宽度/300px
        </Tooltip>
        <Tooltip content="这里是重复的提示文字，这里是重复的提示文字这里是重复的提示文字这里是重复的提示" width={500}>
            自定义宽度/500px
        </Tooltip>
        <Tooltip content="这里是重复的提示文字，这里是重复的提示文字这里是重复的提示文字这里是重复的提示" overlayStyle={{maxWidth: 'unset'}}>

        </Tooltip>
    </div>
</Playground>
